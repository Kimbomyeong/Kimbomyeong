## 💭 Dev Philosophy

<div align="center">

| | |
|:---:|:---|
| 🔒 | **Safety over Accuracy** — 정확도보다 안전성. 틀릴 수 있는 시스템보다 안전하게 실패하는 시스템을 설계합니다. |
| 🏗️ | **System over Model** — 모델보다 시스템. 좋은 모델 하나보다 검증 가능한 구조를 우선합니다. |
| 📏 | **Evaluation over Performance** — 성능보다 평가 체계. 측정할 수 없으면 개선할 수 없습니다. |
| 🔍 | **Define the Problem First** — 문제 정의 먼저. 구현보다 문제의 경계를 먼저 그립니다. |

</div>
